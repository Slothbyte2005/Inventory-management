main.py – Entry point of the application that controls the overall flow of the inventory management system.

Admin_menu.py – Handles administrator operations such as adding, updating, and deleting products.

User_menu.py – Contains functionalities and menu options available to regular users.

Userlogin.py – Manages user authentication and login validation.

Additional_features.py – Includes additional features to enhance system functionality.

login.db – SQLite database file used to store user credentials and inventory records.

Invoice.zip – Stores generated invoices for sales transactions.

Sales.zip – Contains sales-related data and records.

images.zip – Contains images used in the project.

icons.zip – Contains icons used for interface and display purposes.
